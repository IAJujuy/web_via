Formulario mínimo de diagnóstico.

Esta versión no necesita backend para operar: arma el mensaje, registra el intento localmente y abre WhatsApp o correo. Más adelante se conecta a Google Forms, CRM o backend propio.

Al enviar aceptás que VIA use estos datos solo para responder esta consulta comercial. La página de privacidad queda incluida para la etapa inicial.

Después del envío

  1. VIA confirma recepción.
  2. Se valida alcance.
  3. Se envía propuesta o link de pago.
  4. Se inicia el trabajo acordado.
