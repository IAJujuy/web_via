Esta version corrige el orden visual en celular con una modificacion real de HTML, no solo CSS.

Orden fisico en index.html:
1. Logo VIA
2. Avatar CAP VIA
3. Texto, tarjetas y botones

Importante:
Subir a GitHub los archivos de esta carpeta como RAIZ del sitio, reemplazando index.html, cap-via.html, styles.css y assets.
No subir una carpeta contenedora adicional si el sitio ya publica desde la raiz del repositorio.

Tambien se agrego versionado al CSS:
styles.css?v=mobile-logo-avatar-v4
Esto evita que Chrome/GitHub Pages conserve el CSS viejo en cache.


IMPORTANTE: subir tambien la carpeta assets completa. Esta version ademas incluye fallback anti-imagen-rota para logo y avatars, pero assets debe quedar en la raiz del sitio para produccion.
