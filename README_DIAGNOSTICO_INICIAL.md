# VIA Sistemas Digitales - versión corregida de producto

## Qué se corrigió

- Se eliminó la superposición del logo con los títulos en las páginas secundarias.
- Se reemplazaron posiciones absolutas frágiles por una grilla visual estable.
- Se mantuvo el modelo de escritorio sin scroll: cada pantalla está pensada como una sola lámina.
- En celular y pantallas angostas, el sitio pasa a modo vertical con scroll mínimo.
- Se ordenó la jerarquía comercial: diagnóstico inicial, sistemas operativos, identidad digital, agentes IA y CAP VIA.
- Se unificaron botones, tarjetas, espaciado, contraste, foco accesible y textos de acción.

## Archivos principales

- `index.html`: portada comercial.
- `diagnostico.html`: pantalla de diagnóstico inicial.
- `sistemas.html`: pantalla de sistemas operativos.
- `identidad.html`: pantalla de identidad digital.
- `agentes.html`: pantalla de agentes IA.
- `cap-via.html`: pantalla de CAP VIA.
- `styles.css`: diseño unificado y responsive.
- `assets/`: imágenes existentes del proyecto.

## Uso

Abrir `index.html` en el navegador o subir todo el contenido de esta carpeta al repositorio de GitHub Pages.
